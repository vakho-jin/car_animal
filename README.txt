# car_animal
# django car and animal project

GET /car/ - ყველა მანქანის სიის გამოტანა
POST /car/ - ახალი მანქანის დამატება
GET /car/<id>/ - კონკრეტული მანქანის ინფორმაციის გამოტანა აიდის მიხედვით
PUT /car/<id>/ - მანქანაზე ინფორმაციის დამატება (განახლება) აიდის მიხედვით
DELETE /car/<id>/ - მანქანის წაშლა აიდის მიხედვით
GET /animal/ - ყველა ცხოველის სიის გამოტანა
POST /animal/ - ახალი ცხოველის დამატება
GET /animal/<id>/ - კონკრეტული ცხოველის ინფორმაციის გამოტანა აიდის მიხედვით
PUT /animal/<id>/ - ცხოველზე ინფორმაციის დამატება (განახლება) აიდის მიხედვით
DELETE /animal/<id>/ - ცხოველის წაშლა აიდის მიხედვით
